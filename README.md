## üêõ Troubleshooting

### Common Issues

#### Ship Not Detected
- **Solution**: Ensure ship core is properly welded to ship structure
- **Check**: Entities are physically connected within 2000 units
- **Verify**: Only one ship core per ship (no duplicates)
- **Ask ARIA-2**: `!ai ship core` for detailed help

#### Weapons Not Working
- **Solution**: Confirm ship core is present and linked
- **Check**: Weapons are within 2000 units of ship core
- **Verify**: Ship has sufficient energy for weapon operation
- **Ask ARIA-2**: `!ai weapons` for combat system help

#### Stargate Technology Issues
- **Solution**: Use `asc_stargate_info` to check technology compatibility
- **Check**: Technology tiers are compatible (higher can use lower)
- **Verify**: Sufficient energy for higher-tier technologies
- **Ask ARIA-2**: `!ai ancient tech` or `!ai <culture> tech`

#### ARIA-2 AI Not Responding
- **Solution**: Check that AI is enabled in configuration
- **Check**: Use correct command format: `!ai <question>`
- **Verify**: No conflicting chat addons
- **Command**: `asc_ai_status` to check AI system status

### Performance Issues
- **Reduce visual effects**: Lower quality in configuration
- **Optimize updates**: Increase update intervals for better performance
- **Enable optimizations**: Use network optimization features
- **Limit complexity**: Keep ship size and entity count reasonable
